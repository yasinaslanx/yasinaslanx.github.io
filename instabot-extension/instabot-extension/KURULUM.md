# INSTABOT Chrome Extension
## Kurulum Talimatları

### 1. Dosyaları Hazırla
Bu klasörü bilgisayarında bir yere kaydet (örn: `C:\ÇALIŞMALARIM\instabot-extension`)

### 2. Chrome'da Aç
1. Chrome'u aç
2. Adres çubuğuna yaz: `chrome://extensions`
3. Sağ üstte **"Geliştirici modu"** (Developer mode) anahtarını **AÇ**

### 3. Extension'ı Yükle
1. **"Paketlenmemiş öğe yükle"** (Load unpacked) butonuna tıkla
2. Bu klasörü seç (`instabot-extension` klasörü)
3. Extension yüklendi! 🎉

### 4. Kullan
1. instagram.com'a git
2. Kendi profilini aç
3. Sağ üstte 📡 ikonuna tıkla
4. **"ANALİZİ BAŞLAT"** butonuna bas
5. Analiz bitince bildirim gelir
6. Takip etmeyenleri popup'ta görürsün

### Notlar
- Analiz ~16-30 dakika sürer (following sayısına göre)
- Instagram'da giriş yapılı olmak gerekir
- Analiz sırasında sekmeyi kapatma
- Her unfollow turundan sonra 1-2 saat bekle
